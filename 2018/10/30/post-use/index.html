使用Hexo搭建本地博客并git上传

Posted on 2018-10-30

Hexo本地使用

从零开始安装Hexo

git clone https://github.com/iissnan/hexo-theme-next themes/next

hexo clean //清除缓冲
hexo g // 重新生成代码
hexo s // 部署到本地

New repository // 命名为username.github.io

deploy:
type: git
repository: https://github.com/username/username.github.io
branch: master

hexo clean
hexo g
hexo d //hexo deploy

hexo new ‘new_post’ //hexo n ‘new_post’

hexo g
hexo d
`
